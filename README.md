# Interactive Webpage Form

An interactive **web form application** built with **HTML**, **CSS** and **Vanilla JavaScript**, featuring client-side validation and user interaction.

This project was developed as part of my internship tasks at Codveda Technologies, focusing on form handling, validation logic and basic UI interactivity.


## Features

- Interactive form with multiple input types:

  - Full name

  - Email

  - Password

  - Age

  - City (dropdown)

  - Gender (radio buttons)

  - About (textarea)


- Client-side validation using JavaScript:

  - Required field checks

  - Email format validation

  - Password length validation

  - Age range validation

- User feedback via alert on successful submission

- Clean and responsive layout with CSS


## Tech Stack

- **HTML5** – Page structure

- **CSS3** – Styling and responsive layout

- **JavaScript (Vanilla)** – Event handling and form validation logic

- **Git & GitHub** – Version control

- **GitHub Pages** – Live deployment
